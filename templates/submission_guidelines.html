Subtle Differences Recognition Challenge

The challenge consists of two tasks divided into two phases:

1) Difference Image Selection Task

The input for this task consists of two images, along with a text describing the differences between the image pair. The objective is to choose which of the two images contains a change described in the text.

Evaluation Metric:

  • Accuracy (as itâ€™s a binary classification task)

Details od submission format

The submission file should be in JSON format and should contain the image name and the key, and answer("After" or "Before") for each pair. Here is an example:

{
  "name": "test/v1_1010_oboe",
  "attributes": [
      {
          "key": "shape_0",
          "answer": "X"
      },
      {
          "key": "color_1",
          "answer": "X"
      },
      {
          "key": "texture_2",
          "answer": "X"
      },
      {
          "key": "texture_3",
          "answer": "X"
      }
  ]
}

2) Conditional Difference Captioning Task

The goal of this task is to describe the minor differences in shape, color, or texture between two images.

Evaluation Metrics:

  • BLEU-4 (measures the similarity between generated and reference captions)
  • CIDEr (measures the similarity between generated and reference captions)

Submission Format

The submission file should be in JSON format and should contain the images name, and attributes with detailed descriptions. Here is an example:

{
  "name": "test/v1_1010_oboe",
  "attributes": [
      {
          "key": "shape_0",
          "value": "The oboe in image 1 xxxxxxxx."
      },
      {
          "key": "color_1",
          "value": "The oboe in image 1 xxxxxxxx."
      },
      {
          "key": "texture_2",
          "value": "The oboe in image 1 xxxxxxxx."
      },
      {
          "key": "texture_3",
          "value": "The oboe in image 1 xxxxxxxx."
      }
  ]
}

Partiipants are free to participate either in both phases or just 1 of them. The phases will be evaluated separately and winners will be announced for both tasks.
